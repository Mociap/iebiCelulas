1. Estrutura do Projeto

Sistema-Igreja/
├── assets/               # Recursos multimídia
│   ├── ColheitaAmor/    # Imagens Colheita do Amor
│   ├── Coordenadores/   # Fotos coordenadores
│   ├── Lideres/         # Fotos líderes
│   ├── Pastores/        # Fotos pastores
│   └── Supervisores/    # Fotos supervisores
├── Em Andamento/        # Projetos em desenvolvimento

├── Login/               # Sistema de autenticação
├── ministerios/         # Gestão ministerial

├── RotinasPython/      # Scripts de automação
└── Site/               # Frontend principal

Estrutura de Dados - Células

  • nomeigreja: String (texto em maiúsculas)
  • rede: String (texto em maiúsculas)
  • Pastor: String (texto em maiúsculas)
  • pastor1: String (texto em maiúsculas)
  • pastor2: String (texto em maiúsculas)
  • coordenador: String (texto em maiúsculas)
  • coordenador1: String (texto em maiúsculas)
  • coordenador2: String (texto em maiúsculas)
  • supervisor: String (texto em maiúsculas)
  • supervisor1: String (texto em maiúsculas)
  • supervisor2: String (texto em maiúsculas)
  • conclider: String (texto em maiúsculas)
  • nomelider1: String (texto em maiúsculas)
  • nomelider2: String (texto em maiúsculas)
  • nomelider3: String (texto em maiúsculas)
  • nomecelula: String (texto em maiúsculas)
  • diaSemana: String (texto em maiúsculas - ex: "SEGUNDA-FEIRA")
  • horario: String (formato HH:mm - ex: "19:30")
