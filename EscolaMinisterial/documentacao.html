Banco de Dados

Firebase Collections

O sistema utiliza o Firebase como banco de dados NoSQL. Abaixo estão as principais coleções:

Collection: membros

  • arrolamento (string)
  • bairro (string)
  • celular (string)
  • cidade (string)
  • dtnasc (string)
  • estadoCivil (string)
  • igreja (string)
  • lidercelula (string)
  • nome (string)
  • nomedacelula (string)
  • rede (string)
  • sexo (string)
  • uf (string)

Collection: cursos

  • Documento Curso
    • nomeCurso (string)
    • responsavel (string)
    • Subcollection: disciplina
      • Documento Disciplina
        • nomeDisc (string)
        • professor (string)

Collection: cursosInscricoes

  • celula (string)
  • dataInscricao (timestamp)
  • disciplina (string)
  • nome (string)
  • turma (string)

Collection: cursosPresencas

  • dataAula (string)
  • disciplina (string)
  • presencas (map)
  • turma (string)

Collection: cursosTurmas

  • disciplinas (string)
  • dtFim (string)
  • dtIni (string)
  • nomeTurma (string)
  • professor (string)
